MyTravel - Hotels, Flights & Vacations HTML Template

MyTravel is a complete Travel HTML template for hotels, vacation rentals, flights, cars, yachts & activities. It is a stunning looking HTML template using quality fonts and spacing. Our code is developer-friendly, and our documentation is detailed.

Template Features

MyTravel comes with the features highlighted below:

  • 52 HTML templates
  • Cleanly coded and well documented
  • 100% Responsive
  • Based on Bootstrap 4
  • SASS Integrated & Gulp built
  • 10 Unique Home page Layouts
  • Hotels, Listings, Single Page & Booking
  • Tours, Listings, Single Page & Booking
  • Vacation Rentals, Listings, Single Page & Booking
  • Car Rentals, Listings, Single Page & Booking
  • Yachts, Listings, Single Page & Booking
  • Flights, Listings, Single Page & Booking
  • 2-3 listing variations for Hotels, Tours, Vacation Rentals, Car Rentals, Yachts & Flights.
  • Google Fonts & FontAwesome
